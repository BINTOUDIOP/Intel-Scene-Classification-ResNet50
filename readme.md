# Natural scenes classification  intel dataset

Ce dépôt contient une solution de Deep Learning pour la classification de scènes naturelles, utilisant le Transfer Learning pour maximiser les performances.

## Stack technique
- **Framework** : TensorFlow  Keras
- **Architecture** : ResNet50 (Fine-tuned)
- **Traitement** : Data Augmentation (Random Flip, Rotation, Zoom)
- **Dataset** : Intel Image Classification (Kaggle)
